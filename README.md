# RAG-Based YouTube Chatbot

This project is a Retrieval-Augmented Generation (RAG) chatbot that answers questions from YouTube videos.

## Features
- Extract YouTube transcripts
- Vector embeddings
- Semantic search
- AI-generated answers

## Tech Stack
Python, LangChain, FAISS, LLM APIs

## Run Project

pip install -r requirements.txt

python rag_yout_tube_chatbot.py
